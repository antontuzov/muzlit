### Features
- Official Spotify API Use
- Playlists, Playlist Creation,
- Browse & Recommended
- Search Songs, Albums, Artists, More
- Playback and Playlists Playback
- Save Albums
- Sign In/Sign Out (OAUTH 2.0)
- View Your Profile
- Browse Categories
- Categorical Playlists
